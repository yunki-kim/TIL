## 배운 내용 정리

### 병합 정렬
- 재귀용법을 활용한 정렬 알고리즘
- 리스트를 잘라 작은 단위로 만들고 각 리스트를 병합하며 정렬하는 방식
> 시간 복잡도 <br>
> 최악의 경우 : O(nlogn) <br>
<br>

### 퀵 정렬
- 재귀용법을 활용한 정렬 알고리즘
- 기준점을 정해 작은 데이터는 왼쪽, 큰 데이터는 오른쪽에 두고 각 왼쪽, 오른쪽에서 새로운 기준점으로 동일한 방식으로 진행하며 정렬하는 방식
> 시간 복잡도 <br>
> 최악의 경우 : O(n^2) <br>
> 최선의 경우 : O(nlogn) <br>
<br>
