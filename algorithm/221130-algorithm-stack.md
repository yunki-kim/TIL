## 배운 내용 정리

### Stack
- 가장 나중에 들어간 데이터를 먼저 꺼낼 수 있는 후입선출(LIFO) 구조
- 데이터를 제한적으로 접근
- 함수가 동작하는 방식이 Stack 방식
- 데이터 추가 : `push()`
- 데이터 삭제 : `pop()`
- 장점 : 구조가 단순, 데이터 저장/읽기 속도가 빠르다
- 단점 : 데이터의 갯수를 미리 지정해놓아야하므로 저장 공간의 낭비가 발생할 수 있다.
